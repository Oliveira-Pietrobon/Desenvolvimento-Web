Design Responsivo

O Design Responsivo (RWD) visa criar layouts que se ajustam de forma fluida a diferentes tamanhos de tela, utilizando técnicas como media queries, unidades relativas e layouts flexíveis para uma experiência de usuário consistente em qualquer dispositivo.

Principais Técnicas:

Media Queries:Ajusta os estilos de acordo com as características do dispositivo (ex. largura de tela).

Unidades Relativas:Ajusta os estilos de acordo com as características do dispositivo (ex. largura de tela).

Layouts Flexíveis (Flexbox e Grid):Organizam os elementos de forma dinâmica, ajustando-se ao espaço disponível.

Imagens Flexíveis:Usar `max-width: 100%` para que as imagens se ajustem ao tamanho do contêiner.

Mobile First:Desenvolver para dispositivos móveis primeiro e expandir para telas maiores.

Benefícios:

Experiência consistenteem todos os dispositivos.

Manutenção simplificadacom um único código.

Melhoria no SEOjá que o Google favorece sites responsivos

Redução de custos pois um único site atende a todos os dispositivos.

HTML

HTML é a linguagem fundamental para criar a estrutura e o conteúdo de uma página web. Com a utilização de tags, como <div>, <form>, <input>, podemos construir a base de qualquer site.

Principais Componentes:

Tags: Elementos que delimitam e estruturam o conteúdo (ex. h1, p, a).

Atributos:Informações adicionais dentro das tags (ex. href em a href="...")

.

Elementos de Formulário: Usados para coletar dados do usuário (ex. input, textarea).

Em resumo, o HTML forma a base de qualquer página web, estruturando o conteúdo de forma que os navegadores possam exibir corretamente.

CSS

O CSS é responsável pela aparência de uma página web. Utilizando seletores, propriedades e valores, podemos definir estilos como cores, fontes, tamanhos e disposição de elementos, incluindo técnicas avançadas como Flexbox e Grid Layout.

Componentes principais:

Seletores: Identificam os elementos HTML a serem estilizados (ex. h1, .classe, #id).

Propriedades:Definem o estilo a ser aplicado (ex. color, font-size, margin).

Valores:Especificam o valor da propriedade (ex. red, 16px, 10px).

Técnicas Avançadas:

Flexbox:

O Flexbox é um modelo de layout unidimensional que alinha itens em uma linha ou coluna. Ele é ideal para distribuir espaço entre elementos de forma flexível.

Principais propriedades:

display: flex; Ativa o modelo Flexbox.

justify-content:Alinha itens no eixo principal (horizontal).

align-itemsAlinha itens no eixo transversal (vertical).

flex-direction:Define a direção dos itens (linha ou coluna).

flex-wrap:Permite que os itens quebrem para uma nova linha.

Grid Layout:

O CSS Grid é um layout bidimensional, ou seja, trabalha com linhas e colunas ao mesmo tempo. Ele é ideal para layouts mais complexos e precisos.

Principais propriedades:

display:grid;: Ativa o modelo Grid.

grid-template-columns:Define as colunas do grid.

grid-template-rows: Define as linhas do grid.

grid-gap:Define o espaço entre os itens.

grid-column e grid-row:Controlam a posição dos itens nas linhas e colunas.

Comparação:

Flexbox é melhor para layouts unidimensionais (linha ou coluna)

Grid é ideal para layouts bidimensionais (linhas e colunas).

Ambos podem ser usados juntos para criar layouts flexíveis e dinâmicos.

JavaScript

JavaScript é essencial para a interatividade da web. Ele permite que você manipule o DOM, crie animações, validações de formulário e se comunique com servidores, tornando os sites mais dinâmicos e responsivos.

JavaScripté uma linguagem de programação amplamente usada no desenvolvimento web para adicionar interatividade às páginas. Enquanto o HTML define a estrutura do conteúdo e o CSS cuida da aparência, o JavaScript permite que a página “reaja” às ações do usuário, como cliques, digitação ou movimento do mouse.

Ele é executado diretamente no navegador e pode

:

Modificar o conteúdo da página sem recarregá-la (como atualizar uma lista ou mostrar mensagens).

Validar formulários antes do envio.

Criar animações e efeitos visuais.

Comunicar-se com servidores via APIs (por exemplo, usando `fetch` para acessar dados externos).

Controlar eventos e interações em tempo real.

Além disso, JavaScript evoluiu com ferramentas modernas como Node.js (para rodar fora do navegador), React, Vue e TypeScript, tornando-se essencial para o desenvolvimento full stack.

TypeScript

TypeScript é uma linguagem que estende o JavaScript com tipagem estática. Isso permite detectar erros em tempo de compilação e trabalhar com tipos complexos, melhorando a legibilidade e manutenção do código.

Por que usar TypeScript?

Evita erros comuns ao alertar problemas antes da execução.

Facilita a leitura e manutenção do código, especialmente em projetos grandes.

Melhora a produtividade com sugestões de código mais inteligentes em editores como VS Code.

É compatível com JavaScript, ou seja, todo código JS é um código TS válido.

Node.js

Node.js permite a execução de JavaScript no lado do servidor, permitindo o desenvolvimento de back-end com a mesma linguagem usada no front-end. Ele é especialmente útil para criar servidores, APIs e sistemas de tempo real.

Exemplos de uso:

Criar um servidor HTTP. : Com Node.js, é possível criar um servidor que responde a requisições feitas por navegadores ou outras aplicações, como quando você acessa uma página da web.

Desenvolver uma API REST.: Node.js permite criar APIs que enviam e recebem dados no formato JSON, facilitando a comunicação entre o front-end e o back-end.

Conectar a bancos de dados: É possível usar Node.js para se conectar a bancos de dados como MongoDB, MySQL ou PostgreSQL, armazenando e recuperando dados de forma eficiente.

Gerenciar uploads e downloads.

Flexbox

O Flexbox é uma técnica de layout no CSS que permite distribuir espaço entre itens em um contêiner de forma eficiente, mesmo quando o tamanho do conteúdo é desconhecido ou dinâmico.

Propriedades do Contêiner Flex (Pai)

display: flex;Define um contêiner flexível. Os filhos desse contêiner se tornam itens flexíveis.

flex-direction Controla a direção dos itens dentro do contêiner. Os valores podem ser:

row (padrão): Itens ficam na linha (horizontal

).

column:Itens ficam na coluna (vertical).

row-reverse: Itens ficam na linha, mas na ordem invertida.

column-reverse:Itens ficam na coluna, mas na ordem invertida.

flex-wrapControla o comportamento dos itens quando não cabem na linha ou coluna.

nowrap:Todos os itens ficam em uma única linha (padrão).

wrap:Itens podem se mover para a linha seguinte, se necessário.

wrap-reverse:Itens vão para a linha seguinte, mas a ordem é invertida.

justify-contentAlinha os itens ao longo do eixo principal (horizontal por padrão).

flex-start: Alinha os itens ao início.

flex-end:Alinha os itens ao final.

center:Centraliza os itens.

space-between:Espaço igual entre os itens.

space-around: Espaço igual ao redor de cada item

space-evenly:Espaço igual entre os itens e as extremidades.

align-items Alinha os itens ao longo do eixo transversal (vertical por padrão)

flex-start:Alinha no início do contêiner.

flex-end:Alinha no final do contêiner

center:Alinha ao centro.

baseline:Alinha os itens pela linha de base do texto

stretch:Itens se esticam para ocupar toda a altura (padrão).

align-contentControla o alinhamento das linhas do contêiner, quando há várias linhas de itens flexíveis.


Propriedades dos Itens Flex (Filhos)

flex-growDefine a capacidade de um item de crescer para preencher o espaço disponível. O valor padrão é 0 (sem crescimento). Valores maiores significam maior espaço ocupado.

flex-shrinkControla a capacidade de um item encolher quando o espaço for insuficiente. O valor padrão é 1 (pode encolher).

flex-basis Define o tamanho inicial do item antes de ser ajustado pelo flex-grow ou flex-shrink. Pode ser um valor de comprimento (ex. 200px, 20%) ou auto.

flexÉ uma forma abreviada para definir flex-grow, flex-shrink e flex-basis ao mesmo tempo.

align-selfPermite que o item flexível tenha um alinhamento próprio, sobrepondo a configuração de align-items do contêiner.

auto:Usa o valor de align-items do contêiner (padrão).

flex-start, flex-end, center, baseline, stretch: Varia conforme a necessidade.

CSS Grid

CSS Grid é uma poderosa técnica de layout que possibilita criar layouts complexos de forma simples e intuitiva. Ele permite definir linhas e colunas para posicionar elementos de forma precisa.

Propriedades do Contêiner (Pai):

display:grid: Define o contêiner como um grid

.

grid-template-columns Especifica o número e o tamanho das colunas do grid.

grid-template-rows:Define o número e o tamanho das linhas do grid.

grid-template-areas:Permite a definição de áreas nomeadas dentro do grid.

gap:Define o espaçamento entre as colunas e as linhas do grid.

grid-auto-rowsDefine o tamanho das linhas criadas automaticamente.

grid-auto-columns: Define o tamanho das colunas criadas automaticamente.

rid-auto-flow:Controla o fluxo de posicionamento dos itens (linha ou coluna).

Propriedades dos Itens (Filhos):

grid-column:Define a posição de um item nas colunas do grid

grid-row: Define a posição de um item nas linhas do grid.

grid-column-start:Define onde um item começa nas colunas.

grid-column-end: Define onde um item termina nas colunas.

grid-row-start: Define onde um item começa nas linhas.

grid-row-end:Define onde um item termina nas linhas.

grid-area:Combina a definição de linhas e colunas para determinar a área de um item.

Propriedades de Alinhamento:

justify-items: Alinha os itens no eixo horizontal dentro do grid.

align-items:Alinha os itens no eixo vertical dentro do grid.

place-items: Alinha os itens em ambos os eixos (horizontal e vertical).

justify-self: Alinha um item no eixo horizontal, dentro de seu próprio espaço.

Alinha um item no eixo vertical, dentro de seu próprio espaç

Media Queries

Media Queries são utilizadas para aplicar estilos CSS diferentes dependendo das características do dispositivo, como largura da tela, orientação e resolução, sendo fundamentais para o design responsivo.

@media: A regra principal para definir as media queries, permitindo que você aplique estilos condicionais com base nas características do dispositivo.

min-width: Define o estilo quando a largura da tela é maior ou igual ao valor especificado.

max-width: Define o estilo quando a largura da tela é menor ou igual ao valor especificado

.

min-height:Define o estilo quando a altura da tela é maior ou igual ao valor especificado.

max-height:Define o estilo quando a altura da tela é menor ou igual ao valor especificado.

orientation:Permite aplicar estilos com base na orientação da tela, como retrato (portrait) ou paisagem (landscape).

resolution:Permite definir regras para diferentes resoluções de tela, como telas de alta densidade de pixels (dpi ou dppx).

device-width:Aplica estilos com base na largura total da tela do dispositivo (excluindo barras de rolagem ou barras de navegação).

device-height:Aplica estilos com base na altura total da tela do dispositivo.

prefers-color-scheme:Aplica estilos com base na preferência de tema de cor do usuário (claro ou escuro).

hover:Aplica estilos para dispositivos que possuem a capacidade de hover (passar o cursor sobre elementos).

pointer: Define estilos com base na capacidade de apontar do dispositivo, como fine (dispositivo com controle preciso, como um mouse) ou coarse (dispositivo com controle impreciso, como uma tela sensível ao toque).

aspect-ratio:Define a proporção entre a largura e a altura da tela

.
